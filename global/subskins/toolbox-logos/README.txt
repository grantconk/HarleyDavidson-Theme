INSTRUCTIONS
============

To customize the logo at the top-left corner, simply open your userChrome.css file 
from the "Tools > Edit My Config > Edit userChrome.css" dropdown by adding one of 
the below @import lines before the existing @namespace line. 
Then save and restart Firefox.


@import url("chrome://global/skin/subskins/toolbox-logos/conklin.css");
@import url("chrome://global/skin/subskins/toolbox-logos/chapman.css");
